## Project Overview
This project focuses on analyzing **Snitch Fashion sales data** using Microsoft Excel to identify key revenue drivers, customer purchasing patterns, and overall sales performance.  
The dashboard provides actionable insights to support **data-driven business decisions**.

## Application Used
- MS Excel 2016

## Dataset
- Snitch Fashion Sales Data (Excel format)

## Steps Followed
- Imported raw sales data into Excel workbook.
- Performed data cleaning and transformation using:
  - IF
  - AVERAGEIFS
  - DATE
  - DATEVALUE
  - Power Query
- Created Pivot Tables from the cleaned dataset.
- Designed interactive charts:
  - Bar Chart
  - Line Chart
  - Pie Chart
- Built an interactive dashboard using slicers and filters.

## Key Insights Derived
- Standardized discount strategies across cities can increase product sales volume.
- Affordable pricing with controlled margins leads to higher overall revenue.
- Certain cities contribute significantly to total sales and should be targeted for growth.
- Seasonal trends show clear monthly sales fluctuations.

## Conclusion
* The Snitch Fashion Sales Dashboard summarizes historical sales performance using descriptive analysis and supports future planning through revenue forecasting. 
* It provides clear insights into product  performance, regional sales, and trends, enabling data-driven business decisions.
